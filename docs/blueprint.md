# **App Name**: EstimateDecor

## Core Features:

- Floor Area Calculation: Calculate the floor area based on length and width inputs, including material wastage.
- Wall Area Calculation: Calculate total wall area based on individual wall lengths and height.
- Opening Subtraction: Subtract door and window areas from total wall area for accurate paint/wallpaper estimation.
- Ceiling Area Calculation: Calculate ceiling area based on length and width inputs.
- Curtain Measurement Calculation: Calculate optimal curtain width and height based on window dimensions.
- Panel Estimation: Estimate the number of panels needed based on area and panel dimensions.
- Paint Estimation: Estimate the amount of paint needed based on total area and number of coats.

## Style Guidelines:

- Primary color: Orange (#FF7800) for a modern, vibrant feel.
- Background color: White (#FFFFFF), providing a clean and spacious backdrop.
- Accent color: Black (#000000), used for text, icons, and key interactive elements to create a strong contrast.
- Body and headline font: 'Roboto', a modern, readable font suitable for both headlines and body text.
- Clean and intuitive layout with clear sections for each calculation type.
- Simple, elegant icons representing each calculation type (floor, wall, ceiling, etc.).
